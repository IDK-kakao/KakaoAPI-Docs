Installation ​

Setup KakaoAPI in your project.

Requirements ​

  • Windows
  • Visual Studio
  • Brain

Quick Setup ​

1. Get the Files ​

bash
# Clone
git clone Idk lol
1
2

2. Add to Project ​

  1. Copy KakaoAPI.h, KakaoWrapper and libs folder to your project
  2. Include:
cpp
#include "KakaoAPI.h"
1

or high level api -

cpp
#include "KakaoWrapper.h"
1

4. Link Libraries ​

Add to linker dependencies:

cpp
#pragma comment(lib, "KakaoAPI.lib")
1

Project Settings ​

Visual Studio Configuration ​

  1. C++ Standard: Set to C++17 (/std:c++17)
  2. Platform: Use x64 for modern games
  3. Runtime Library:
    • Static: /MT (recommended)
    • Dynamic: /MD
  4. Build config:
    • Release

Test Installation ​

cpp
#include "KakaoAPI.h"
#include <iostream>

#pragma comment(lib, "KakaoAPI.lib")

int main() {
    if (!Kakao::KakaoAPI::Initialize("GameAssembly.dll", true)) {
        std::cerr << "Some err here :(" << std::endl;
        return -1;
    }

    Kakao::Console::SetTitle("KakaoAPI Test");
    Kakao::Console::LogSuccess("Done");
    // Unload
    Kakao::KakaoAPI::Shutdown();
    return 0;
}

Troubleshooting ​

Common Issues ​

"Cannot find MinHook.h"

  • Check include paths in project settings
  • Verify MinHook installation

"Unresolved external symbol"

  • Ensure all libraries are linked
  • Check library paths

Access violation on startup

  • Run game as administrator
  • Verify correct module name
  • Check for x64/x86

Debug Tips ​

cpp
// Enable verbose logging
Kakao::KakaoAPI::SetLogLevel(3);

// Check initialization
if (!Kakao::KakaoAPI::IsInitialized()) {
    // Handle failure
}
