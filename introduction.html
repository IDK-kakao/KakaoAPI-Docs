Introduction ​

KakaoAPI is a C++ framework for hooking and manipulating Unity IL2CPP games.

What is KakaoAPI? ​

A simple library that lets you:

  • Hook Unity methods by name or memory address
  • Read and modify game memory
  • Access Unity's IL2CPP runtime
  • Debug with built-in console logging

Core Modules ​

ModulePurpose
CoreFramework initialization and management
ConsoleDebug logging and output

Basic Usage ​

cpp
#include "KakaoAPI.h"

int main() {
    // Initialize
    Kakao::KakaoAPI::Initialize("GameAssembly.dll", true);
    
    // Hook a method
    void* original = nullptr;
    Kakao::MethodHook::HookMethod(
        "PlayerListGetterHook", "Assembly-CSharp", "Photon.Pun", "PhotonNetwork", "get_PlayerList", 0,
        MyDetour, &original
    );
    
    // Enable hook
    Kakao::MethodHook::Enable("PlayerListGetterHook");
    
    // Unload
    Kakao::KakaoAPI::Shutdown();
    return 0;
}
