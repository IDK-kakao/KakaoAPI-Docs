Console API ​

Debug console for logging and output.

Overview ​

The Console module provides colored logging output for debugging Unity game mods.

Methods ​

MethodDescription
Allocate()Create console window
Free()Close console window
SetTitle(title)Set window title
Log(message)Print info message (white)
LogError(message)Print error message (red)
LogSuccess(message)Print success message (green)

Basic Usage ​

cpp
#include "KakaoAPI.h"

int main() {
    // Console is automatically created by KakaoAPI::Initialize()
    Kakao::KakaoAPI::Initialize("GameAssembly.dll", true);
    
    // Set custom title
    Kakao::Console::SetTitle("My Game Mod");
    
    // Log messages with colors
    Kakao::Console::Log("Mod loaded successfully");
    Kakao::Console::LogSuccess("Hook created!");
    Kakao::Console::LogError("Failed to find method");
    
    return 0;
}

Using with Wrapper ​

cpp
#include "KakaoWrapper.h"
using namespace KakaoWrapper;

int main() {
    // Console automatically created
    KakaoUnityAPI::Init("GameAssembly.dll", true);
    
    // Simplified logging
    KakaoUnityAPI::Log("Info message");
    KakaoUnityAPI::Success("Success message");
    KakaoUnityAPI::Error("Error message");
    
    return 0;
}

Manual Console Management ​

cpp
// Create console manually
Kakao::Console::Allocate();
Kakao::Console::SetTitle("Debug Console");

// Use console
Kakao::Console::Log("Manual console created");

// Clean up
Kakao::Console::Free();

Best Practices ​

1. Use Appropriate Log Levels ​

cpp
// Info for general messages
Kakao::Console::Log("Player count: " + std::to_string(count));

// Success for confirmations
Kakao::Console::LogSuccess("Hook enabled");

// Error for failures
Kakao::Console::LogError("Failed to initialize");

3. Use String Formatting ​

// Format numbers and variables
float health = 75.5f;
Kakao::Console::Log("Player health: " + std::to_string(health));

// Multiple values
int level = 10;
std::string name = "Player1";
Kakao::Console::Log("Player " + name + " reached level " + std::to_string(level));
